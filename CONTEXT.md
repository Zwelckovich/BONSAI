# BONSAI Methodology

Glossary for the BONSAI meta-repo — the methodology, rules, and skills that govern all BONSAI projects.

## Language

**State of the art (BONSAI)**:
The latest stable major version of an actively maintained tool, verified against live sources at audit time. A beta qualifies only with a documented escape hatch to a stable alternative (precedent: ty with the pyrefly hatch).
_Avoid_: modern, current, up-to-date (unqualified)
